Based on the amount of [[Resources-Data/Knowledge/Science/Chemistry/Solutions/Solution, Solute, and Solvent#Solute\|solute]] dissolved, solutions can be classified into three types:

+--------------------------------+-------------------------------------------------------------------------------------------------+
| 1. **Unsaturated Solution**    | Contains ==less solute== than the maximum amount that can be dissolved at a given temperature.  |
|                                |                                                                                                 |
|                                | More solute can still dissolve in it.                                                           |
+--------------------------------+-------------------------------------------------------------------------------------------------+
| 2. **Saturated Solution**      | Contains the ==maximum amount of solute== that can dissolve at a given temperature.             |
|                                |                                                                                                 |
|                                | Any additional solute will no longer dissolve.                                                  |
+--------------------------------+-------------------------------------------------------------------------------------------------+
| 3. **Supersaturated Solution** | Contains ==more solute== than what should normally dissolve at a given temperature.             |
|                                |                                                                                                 |
|                                | It is unstable and can easily crystallize when disturbed.                                       |
+--------------------------------+-------------------------------------------------------------------------------------------------+

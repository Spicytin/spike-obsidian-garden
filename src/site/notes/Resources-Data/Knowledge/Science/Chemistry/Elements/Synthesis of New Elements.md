+---------------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| 1. **Particle Accelerators**          | These machines speed up charged particles to ==extremely high velocities==. When these particles hit a target nucleus, they may combine, producing a new element with a different atomic number. |
|                                       |                                                                                                                                                                                                  |
|                                       | E.g: ***Curium***-246 + alpha particle $\rightarrow$ ***Berkelium***-249                                                                                                                         |
+---------------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| 2. **Neutron Capture and Beta Decay** | Some elements are created by capturing **[[Resources-Data/Knowledge/Science/Chemistry/Atoms#Main Parts of an Atom\|neutrons]]**, which later convert into **[[Resources-Data/Knowledge/Science/Chemistry/Atoms#Main Parts of an Atom\|protons]]**.                                       |
|                                       |                                                                                                                                                                                                  |
|                                       | E.g: ***Uranium*** absorbs a neutron $\rightarrow$ becomes ***Neptunium*** (*Z = 93*) $\rightarrow$ decays to ***Plutonium*** (*Z = 94*)                                                         |
+---------------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| 3. **Nuclear Fusion**                 | Two lighter nuclei collide and fuse to form a ==heavier nucleus==.                                                                                                                               |
|                                       |                                                                                                                                                                                                  |
|                                       | E.g: ***Calcium***-48 fused with ***Californium***-249 $\rightarrow$ ***Organesson***-118, one of the heaviest synthetic elements.                                                               |
+---------------------------------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

The creation of new elements has allowed scientists to:
* Expand the **[[Resources-Data/Knowledge/Science/Chemistry/Periodic Table\|periodic table]]** beyond naturally occurring elements.
* Study nuclear stability and decay.
* Develop new materials used in medicine, industry, and research.
* Understand the forces inside atomic nuclei.

Synthetic elements like ***technetium*** are used in medical imaging: while ***americium*** is used in smoke detectors.
